
# RV64GCB+ Instruction Listing

Arch: RV64IMAFDCB\_Zicsr\_Zifencei\_Zfa\_Zicond\_Zihintpause\_Zawrs\_Zacas


## 16-Bit Instructions

Registers marked with a `'` suffix only use 3 bits for their encoding and refer
to the restricted set (i.e. x8 - x15 or f8 - f15, etc).

Registers not marked with `'` use the regular 5 bits and can refer to any
register.
